# HILEX Landing Page

This is the public marketing website for:

```text
https://hilex.co.uk
```

It is separate from the members platform:

```text
https://membersaccess.hilex.co.uk
```

The package buttons submit directly to:

```text
https://membersaccess.hilex.co.uk/api/stripe/checkout
```

After payment, Stripe returns members to the members platform so they can set their password and access the library.
